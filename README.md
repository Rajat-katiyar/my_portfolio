# Portfolio (Next.js)

A modern portfolio website built with Next.js, React, and TypeScript.

## Features

- Responsive design
- Modern UI/UX
- Smooth animations
- Fast performance with Next.js
- TypeScript support

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Scripts

```bash
npm run dev     # start development server
npm run build   # production build
npm run start   # start production server
npm run lint    # run linter
```

## Structure

- `app/`: Next.js App Router pages and layout
- `components/`: React components
- `data/`: Project data and other static data
- `public/`: Static assets (images, etc.)
- `src/style.css`: Global styles

## Customize

- Update text in `components/Hero`, `components/About`, and `components/Contact`
- Edit `data/projects.ts` to add your own projects
- Tweak styles in `src/style.css`
- Update `app/layout.tsx` for metadata changes
